## **Q4 (b)**

**Query Builder Method to fetch all posts with user names (CodeIgniter style)**

ЁЯСЙ Assume:

* `users(id, name)`
* `posts(id, title, user_id)` тЖТ `user_id` is foreign key referencing `users.id`

### ЁЯФ╣ CodeIgniter Query Builder Code

```php
$this->db->select('posts.*, users.name');
$this->db->from('posts');
$this->db->join('users', 'users.id = posts.user_id');
$query = $this->db->get();
$result = $query->result();
```
