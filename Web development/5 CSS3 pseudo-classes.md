```html
<!DOCTYPE html>
<html>
<head>
    <title>CSS3 Pseudo Classes</title>
    <style>
        /* 1. :hover */
        button:hover {
            background-color: green;
            color: white;
        }

        /* 2. :active */
        button:active {
            background-color: red;
        }

        /* 3. :focus */
        input:focus {
            border: 2px solid blue;
            background-color: #e6f2ff;
        }

        /* 4. :first-child */
        ul li:first-child {
            color: purple;
            font-weight: bold;
        }

        /* 5. :nth-child() */
        ul li:nth-child(2) {
            color: orange;
        }
    </style>
</head>
<body>

    <h2>CSS3 Pseudo Classes Demo</h2>

    <!-- Hover & Active -->
    <button>Click Me</button>

    <br><br>

    <!-- Focus -->
    <input type="text" placeholder="Enter your name">

    <br><br>

    <!-- First-child & Nth-child -->
    <ul>
        <li>First Item</li>
        <li>Second Item</li>
        <li>Third Item</li>
    </ul>

</body>
</html>
```

| Pseudo Class    | Use                                      |
| --------------- | ---------------------------------------- |
| `:hover`        | Applies style when mouse is over element |
| `:active`       | Applies when element is clicked          |
| `:focus`        | Applies when input is selected           |
| `:first-child`  | Selects first child element              |
| `:nth-child(n)` | Selects element based on position        |
