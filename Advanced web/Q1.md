# Q1(a) Asynchronous Operations in Node.js

Sabse pehle concept samjho.

## Synchronous

Ek kaam khatam → fir dusra kaam

Example:

```text
Read File
↓
Write File
↓
Delete File
```

Pehla complete hone tak dusra start nahi hota.

---

## Asynchronous

Kaam start karo aur aage badho.

Example:

```text
Read File Start
Write File Start
Delete File Start
```

Node.js wait nahi karta.

Isi ko Non-Blocking bolte hain.

---

## Exam Code

```js
const fs = require('fs');

fs.readFile('data.txt', 'utf8', (err, data) => {
    console.log(data);
});

fs.writeFile('new.txt', 'Hello World', (err) => {
    console.log("File Written");
});

fs.appendFile('new.txt', ' Welcome', (err) => {
    console.log("Data Added");
});

fs.unlink('new.txt', (err) => {
    console.log("File Deleted");
});
```

---

## Exam Me Theory

The File System module provides asynchronous methods such as readFile(), writeFile(), appendFile(), and unlink(). These methods execute without blocking the main thread, allowing Node.js to handle multiple operations efficiently.

---

# Recall

Question:

Node.js asynchronous kyu kehlata hai?

Answer:

Kyuki ye operations complete hone ka wait nahi karta aur non-blocking architecture use karta hai.

# Q1(b) Arrow Function Reverse String

Pehle Arrow Function samjho.

Normal Function:

```js
function add(a,b){
 return a+b;
}
```

Arrow Function:

```js
const add = (a,b) => a+b;
```

Short syntax.

---

## Reverse String Logic

String:

```text
HELLO
```

Convert into Array

```text
H E L L O
```

Reverse

```text
O L L E H
```

Join

```text
OLLEH
```

---

## Exam Code

```js
const reverseString = str =>
    str.split('').reverse().join('');

console.log(reverseString("HELLO"));
```

Output:

```text
OLLEH
```

---

# Recall

String ko array me convert karne ke liye?

```js
split('')
```

Reverse karne ke liye?

```js
reverse()
```

# Q1(c) Path Module

Ye bahut important hai.

## Problem

Path:

```text
C:/Users/Rahul/file.txt
```

Isme se:

* file name chahiye
* extension chahiye
* folder name chahiye

To Path Module use hota hai.

---

## Important Methods

### basename()

```js
path.basename('/home/test/file.txt')
```

Output

```text
file.txt
```

---

### dirname()

```js
path.dirname('/home/test/file.txt')
```

Output

```text
/ home / test
```

---

### extname()

```js
path.extname('file.txt')
```

Output

```text
.txt
```

---

### join()

```js
path.join('users','rahul','docs')
```

Output

```text
users/rahul/docs
```

---

## Exam Answer

The Path module is a built-in Node.js module used for handling file and directory paths. It provides methods such as basename(), dirname(), extname(), and join() to manipulate paths efficiently.

---

# Recall

File extension nikalne ke liye?

**extname()**

Path combine karne ke liye?

**join()**

# Q1(d) Prime Number Program

Pehle concept.

Prime Number:

Sirf 2 factors.

Example:

```text
2
3
5
7
11
13
```

Not Prime:

```text
4
6
8
9
10
```

---

## Logic

Agar number 2 se lekar n-1 tak kisi se divide ho gaya

→ Not Prime

Warna

→ Prime

---

## Exam Code

```ts
let num:number = 13;
let prime:boolean = true;

for(let i=2;i<num;i++){
    if(num%i===0){
        prime=false;
        break;
    }
}

if(prime)
    console.log("Prime Number");
else
    console.log("Not Prime Number");
```
