# Steps to Concept: Question Bank Module

## Concepts

1. **Question Management**: CRUD operations for questions
2. **Categorization**: Organizing questions by topics
3. **Bulk Operations**: Efficient batch processing
4. **Question Types**: Supporting multiple formats

## Algorithmic Thinking

### Question Creation Algorithm
```
ALGORITHM: CreateQuestion
1. VALIDATE question data
2. VALIDATE question type
3. CREATE question record
4. CREATE options (if MCQ)
5. RETURN question
END
```
