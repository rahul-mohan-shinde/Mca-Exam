### 🚀 How to Test

1. **Start Project:**
   ```bash
   START_PROJECT.bat
   ```

2. **Register Account:**
   - Go to http://localhost:4200/register
   - Create account

3. **Login:**
   - Use credentials to login
   - Redirected to dashboard

4. **Create Questions:**
   - Go to /questions
   - Create MCQ, True/False, Short Answer questions

5. **Create Exam:**
   - Go to /exams
   - Create exam
   - Generate question paper

6. **Take Exam (as Student):**
   - Login as student
   - Go to /student
   - Start exam
   - Answer questions
   - Submit exam

7. **View Results:**
   - After submission
   - View result with grade
   - Review answers
